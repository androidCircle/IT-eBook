# IT-eBook App

------

该项目基于 IT-eBook api 并开源，同时使用到Rx, Glide, Retrofit 等开源技术，UI设计基于Material Design的风格， 目前提供预置类别书籍的下载功能，同时将继续优化功能并提供自定义搜索书籍下载
